On the Subject of Synchronization

Bringing order to chaos, one blinking light at a time.

Some lights on the module will flash at different speeds rated 1-5, slowest to fastest. Synchronizing the flashing lights to each other and then to the bomb itself will disarm the module.

Finding the syncing order

Using the displayed number and fastest flashing light position to find the starting cell on the chart. Shift on the chart a number of times based on the speed of the central light in the direction of the slowest light relative to the center light. Wrap around the chart if you need to.

Syncing lights togther

Select a light based on the order you found in the last step. After you select the first light it will stop flashing. Then select the next light to sync with the first light. You can only select lights that haven't been synced yet until there isn't a pair of unsynced lights left.

Syncing with the bomb

After you synced all the lights together, you need to press the button labeled "SYNC" when the number displayed is in the timer.

Asc/Desc = Ascending/Descending order.
Opp = Opposite speed order. Select 5 with 1 or 1 with 5 first and remain consistant in subsequent syncs.
Example: 1 with 5 while on, then 2 with 4 while on. 		+/- = On/Off state.
Alt = Alternating state. Select on or off for the first sync, but subsequent syncs must be opposite states.
Example: 5 with 4 while off, then 3 with 2 while on.
